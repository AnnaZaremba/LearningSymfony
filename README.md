LearningSymfony
===============

Installation
------------

1. Get project
```
git clone https://github.com/AnnaZaremba/LearningSymfony.git
```
2. Install dependencies
```
composer install
```
3. Permission to folders
```bash
chmod -R 0777 var/logs
chmod -R 0777 var/cache
chmod -R 0777 var/sessions
```
